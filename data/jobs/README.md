# Job Applications

Each job you apply to gets its own folder here, created automatically by `/tailor-resume`.

## Folder structure

```
jobs/
├── google-lead-gpm-2026-02-11/
│   ├── posting.md           # Saved job description
│   ├── resume.md            # Tailored resume
│   └── cover-letter.md      # Cover letter
├── stripe-head-growth-2026-02-15/
│   ├── posting.md
│   ├── resume.md
│   └── cover-letter.md
└── ...
```

Folders are named `[company]-[role-slug]-[date]` and contain everything for that application.
